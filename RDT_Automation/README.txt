This Framework will used to automate RDT Automation for Android and IOS Device 

Used Resources  : -
  > JDK 1.8 or above ( Env for Run and Compile )
  > Ant Tool ( flow Automate )
  > Selenium 
  > TestNG ( Suite runner )
  > Log4J ( Log dump )
  > Apache POI ( Excel Sheet Data Reader)
  > DataProvider ( Class Level )
  > Listener Class 
  > Command CLI ( Apache tool)
  > Ashot Jar ( To capture full screen size Image)
